## 🌍 English

![UI Preview](link_to_your_screenshot.png)

**KHA-DIVERGENT** is a conceptual intelligent city infrastructure management system (Smart City), developed for the STEMNOVA hackathon (SmartScape track). The project combines road traffic analysis and thermographic building audits into a single unified cyber-minimalist dashboard.

### 🎯 Problem and Solution
Modern metropolises face two critical infrastructure challenges: massive heat loss during winter seasons and inefficient road traffic logistics. 
KHA-DIVERGENT addresses these issues through two active analytical layers:
1. **Thermographic Layer:** Evaluates building heat loss. Utilizing a built-in AI auditor (LLM), it calculates the required insulation thickness, estimates CO2 emission reduction, and projects annual financial savings.
2. **Traffic Flow Layer:** Simulates intersection congestion and offers automated traffic light phase coordination (creating a "green wave") to reduce bottlenecks and lower vehicle emissions.

### ⚙️ Technology Stack
* **Backend:** Python, FastAPI (WebSockets for real-time telemetry simulation)
* **AI / ML Engine:** Gemini LLM API
* **Frontend:** HTML/CSS/JS (Custom Vector Engine, Procedural UI)
* **Data Processing:** Pandas, NumPy
